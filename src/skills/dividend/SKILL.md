# 配当分析 Skill

配当の水準・トレンド・持続性を多角的に分析し、インカム投資の判断材料を提供する。配当政策の変化はISQシグナルとして反映する。

## 必要なツールとデータ

- `get_financials`: 決算データ（EPS、配当金、純利益、自己資本等）
- `get_key_ratios`: 配当利回り、配当性向等
- Premium プランで詳細な配当履歴が取得可能。Free/Light の場合は Web 検索（TDnet、みんかぶ、IR BANK等）でフォールバック
- 最低プラン: Free（Premium 推奨）

## Workflow Checklist

```
Dividend Analysis Progress:
- [ ] Step 1: 配当データ取得
- [ ] Step 2: 配当利回り推移の分析
- [ ] Step 3: 連続増配/減配年数の特定
- [ ] Step 4: DOE（株主資本配当率）の算出
- [ ] Step 5: 配当性向の分析
- [ ] Step 6: 配当政策の変化シグナル検出
- [ ] Step 7: 結果出力
```

## Step 1: 配当データ取得

`get_financials` と `get_key_ratios` で以下を取得：

**必要データ:**
- 年間配当金（1株あたり）: 過去5〜10年分
- EPS（1株あたり利益）: 配当性向の算出に使用
- 自己資本（純資産 - 新株予約権 - 非支配株主持分）: DOE算出に使用
- 発行済株式数: 1株あたり指標の検証用
- 現在の株価: 配当利回り算出用

**フォールバック:** API で十分なデータが取得できない場合：
- IR BANK（https://irbank.net/）で配当履歴を検索
- みんかぶ（https://minkabu.jp/）で配当利回り情報を取得
- TDnet で適時開示（配当予想の修正等）を確認

## Step 2: 配当利回り推移の分析

```
配当利回り(%) = 年間配当金 / 株価 × 100
```

**確認ポイント:**
- 現在の配当利回り
- 過去5年の配当利回りレンジ（最高/最低/平均）
- 現在の利回りが過去レンジのどの位置にあるか

**利回りの水準判定:**

| 配当利回り | 判定（日本株基準） |
|-----------|-----------------|
| 4.0%超 | 高配当（持続性の確認必須） |
| 3.0〜4.0% | やや高配当 |
| 2.0〜3.0% | 平均的 |
| 1.0〜2.0% | やや低配当 |
| 1.0%未満 | 低配当（成長株の可能性） |

## Step 3: 連続増配/減配年数の特定

過去の年間配当金の推移から：
- **連続増配年数**: 何年連続で配当を増やしているか
- **連続減配年数**: 何年連続で配当を減らしているか
- **据え置き年数**: 変更なしの年数
- **記念配当・特別配当の有無**: 一時的な増配は連続増配にカウントしない

**日本株の増配トレンド:**
- 10年以上連続増配: 非常に優秀（日本では稀）
- 5〜9年連続増配: 優秀
- 3〜4年連続増配: 良好
- 増配基調だが途中据え置きあり: まずまず

## Step 4: DOE（株主資本配当率）の算出

```
DOE(%) = 年間配当総額 / 自己資本 × 100
```

または

```
DOE(%) = 配当性向 × ROE / 100
```

**DOEの意義:**
- 配当性向は利益変動で不安定だが、DOEは自己資本ベースで安定性が高い
- 近年の日本企業はDOEを配当政策の基準に採用する例が増加
- DOE 3%以上を目標とする企業が多い

**DOEの判定:**

| DOE | 判定 |
|-----|------|
| 4.0%超 | 高還元（持続性を確認） |
| 3.0〜4.0% | 還元積極的 |
| 2.0〜3.0% | 平均的 |
| 2.0%未満 | 還元控えめ |

## Step 5: 配当性向の分析

```
配当性向(%) = 年間配当金 / EPS × 100
```

**配当性向の判定:**

| 配当性向 | 判定 |
|---------|------|
| 80%超 | 危険水域（減配リスク高い） |
| 50〜80% | やや高め（利益悪化時のバッファ少ない） |
| 30〜50% | 適正範囲 |
| 30%未満 | 余力あり（増配余地大きい） |

**注意:** 配当性向は一時的な利益変動で大きくブレる。複数年の平均で判断する。

## Step 6: 配当政策の変化シグナル検出

以下の変化はISQシグナルとして検出する：

| シグナル | 内容 | ISQ影響 |
|---------|------|--------|
| 増配発表 | 配当予想の上方修正 | ポジティブ |
| 減配発表 | 配当予想の下方修正 | ネガティブ（大） |
| DOE目標の新設/引き上げ | 配当政策の強化 | ポジティブ |
| 自社株買い + 増配 | 総還元強化 | ポジティブ（強） |
| 配当性向の目標変更 | 還元方針の転換 | 内容による |
| 記念配当の発表 | 一時的な増配 | 中立（一時的） |

**検出方法:** `get_financials` の配当予想データ、または Web検索で直近のIR情報を確認。
