# 同業他社比較 Skill

指定銘柄をセクター内の競合企業と主要財務指標で比較し、相対的なポジションを明らかにする。ISQスコアは算出しない（比較テーブルの提供のみ）。

## 必要なツールとデータ

- `company_screener`: セクター内の同業他社をリストアップ
- `get_key_ratios`: 各銘柄の財務指標（PER, PBR, ROE, ROIC, 配当利回り等）
- `get_financials`: 売上成長率等の補完データ
- 最低プラン: Free（Light 推奨で比較精度向上）

## Workflow Checklist

```
Peer Comparison Progress:
- [ ] Step 1: 対象銘柄のセクター・業種特定
- [ ] Step 2: 同業他社のリストアップ
- [ ] Step 3: 各銘柄の指標取得
- [ ] Step 4: 比較テーブル作成
- [ ] Step 5: 結果出力
```

## Step 1: 対象銘柄のセクター・業種特定

`get_key_ratios` または `get_financials` で対象銘柄のセクター・業種を取得する。

**注意:** 日本株の業種分類は東証33業種を基本とする。グローバル分類（GICS等）が取得される場合は、東証分類に読み替えて同業他社を選定。

## Step 2: 同業他社のリストアップ

`company_screener` で同一セクター/業種の銘柄をリストアップ。

**選定基準:**
- 同一業種の銘柄を時価総額順に取得
- 時価総額が対象銘柄の0.3倍〜3倍の範囲を優先（規模感が近い企業）
- 比較対象は5〜10銘柄を目安

**大手企業の場合:** 業界のリーダー企業を含める（時価総額の倍率に関わらず）

## Step 3: 各銘柄の指標取得

`get_key_ratios` で以下の指標を取得。

**比較指標:**

| 指標 | 説明 | 良い方向 |
|------|------|---------|
| PER（株価収益率） | 利益に対する株価の割高/割安 | 低いほど割安（ただし成長率を考慮） |
| PBR（株価純資産倍率） | 純資産に対する株価の評価 | 1.0以上が望ましい（TSE改革圧力） |
| ROE（自己資本利益率） | 資本効率 | 高いほど効率的（10%以上が目安） |
| ROIC（投下資本利益率） | 事業の本質的な収益力 | WACC以上なら価値創造 |
| 配当利回り | 株価に対する配当の割合 | インカム重視なら高いほど良い |
| 売上成長率（YoY） | 前年同期比の売上成長 | 高いほど成長力あり |
| 営業利益率 | 本業の収益力 | 高いほど稼ぐ力が強い |

## Step 4: 比較テーブル作成

**テーブルのルール:**
- カラム幅を統一し、数値は右寄せで整理
- セクター中央値を算出して比較基準とする
- 対象銘柄のセクター内パーセンタイル（上位何%）を算出

**パーセンタイルの算出:**
各指標について、セクター内での順位から算出。ROE/ROIC/成長率は高い方が上位、PERは低い方が上位。
