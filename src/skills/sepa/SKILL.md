# SEPA & Stage Analysis Skill

Mark Minervini のトレンドテンプレート（SEPA: Specific Entry Point Analysis）7条件の充足チェックと、Stan Weinstein のステージ分類を行う。

## 必要なツールとデータ

- `get_stock_price`: 過去1年以上のOHLCVデータ（SMA200算出に必要）
- `get_technical_indicators`: SMA50, SMA150, SMA200, 52週高値/安値（利用可能な場合）
- 最低プラン: Light（1年以上の日足データが必要）

## Workflow Checklist

```
SEPA Analysis Progress:
- [ ] Step 1: データ取得（1年以上の日足OHLCV）
- [ ] Step 2: 移動平均線の算出（SMA50, SMA150, SMA200）
- [ ] Step 3: 52週高値・安値の特定
- [ ] Step 4: SEPA 7条件の判定
- [ ] Step 5: ステージ分類（S1〜S4）
- [ ] Step 6: 結果出力
```

## Step 1: データ取得

`get_stock_price` で過去260営業日以上（約1年）の日足OHLCVデータを取得する。

`get_technical_indicators` が利用可能な場合は SMA50, SMA150, SMA200 を直接取得。利用不可の場合は Step 2 で手計算する。

## Step 2: 移動平均線の算出

OHLCVデータから以下を算出（`get_technical_indicators` で取得済みならスキップ）：

- **SMA50**: 直近50日終値の単純平均
- **SMA150**: 直近150日終値の単純平均
- **SMA200**: 直近200日終値の単純平均
- **SMA200の傾き**: 20営業日前のSMA200と現在のSMA200を比較。上昇していれば「上向き」

## Step 3: 52週高値・安値の特定

- **52週高値**: 過去260営業日の最高値（高値ベース）
- **52週安値**: 過去260営業日の最安値（安値ベース）
- **現在値の52週高値からの距離**: `(現在値 - 52週高値) / 52週高値 × 100`
- **現在値の52週安値からの距離**: `(現在値 - 52週安値) / 52週安値 × 100`

## Step 4: SEPA 7条件の判定

各条件を Pass / Fail で判定する。

| # | 条件 | 判定基準 |
|---|------|---------|
| 1 | 現在値 > SMA150 かつ 現在値 > SMA200 | 株価が長期MAの上にある |
| 2 | SMA150 > SMA200 | 中期MAが長期MAの上 |
| 3 | SMA200 が上向き（少なくとも1ヶ月） | 20営業日前のSMA200 < 現在のSMA200 |
| 4 | SMA50 > SMA150 かつ SMA50 > SMA200 | 短期MAが中長期MAの上 |
| 5 | 現在値 > SMA50 | 株価が短期MAの上 |
| 6 | 現在値が52週安値から25%以上高い | `(現在値 - 52週安値) / 52週安値 >= 0.25` |
| 7 | 現在値が52週高値の25%以内 | `(52週高値 - 現在値) / 52週高値 <= 0.25` |

**SEPAスコア = Pass数 / 7**

## Step 5: ステージ分類

Stan Weinstein の4ステージモデルで現在のステージを判定する。

| Stage | 名称 | 判定基準 |
|-------|------|---------|
| S1 | 底固め（Accumulation） | SMA200 横ばい or やや下向き。株価がSMA200付近で横ばい。SEPAスコア 0-2/7 |
| S2 | 上昇（Advancing） | SMA200 上向き。株価 > SMA200。SEPAスコア 5-7/7 |
| S3 | 天井形成（Distribution） | SMA200 上向きだが傾き鈍化。株価がSMA200を行ったり来たり。SEPAスコア 3-5/7 |
| S4 | 下落（Declining） | SMA200 下向き。株価 < SMA200。SEPAスコア 0-2/7 |

**判定フロー:**
1. SMA200の傾きを確認（上向き / 横ばい / 下向き）
2. 株価とSMA200の位置関係を確認
3. SEPAスコアを参照して総合判定

**S1 vs S4 の見分け方:**
- S1: SMA200の下落が止まりつつある（傾き鈍化 → 横ばい）。出来高が徐々に増加
- S4: SMA200が明確に下向き。出来高は減少傾向 or 売り出来高が多い

### ISQスコアへの反映

- SEPA 7/7 + S2 → 強いトレンド銘柄。テクニカルISQ加点
- SEPA 5-6/7 + S2 → 概ね良好だが一部条件未達。要確認
- S3移行の兆候（スコア低下中）→ 手仕舞いシグナルとして警告
- S1でスコア上昇中 → 初動の可能性。ウォッチリスト追加候補
